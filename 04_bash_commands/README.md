# 04 Bashコマンド実行

Claude Codeのターミナルコマンド実行機能を体験する。

## お題

1. **テスト実行**: 「test_sample.pyのテストを実行して」
2. **依存インストール**: 「requirements.txtのパッケージをインストールして」
3. **スクリプト実行**: 「hello.shを実行して」
4. **プロセス確認**: 「Pythonが動いているプロセスを確認して」

## 特徴

- 破壊的操作（rm -rf等）は実行前に確認を求める
- ファイル操作はBashではなく専用ツール（Read/Write/Edit）を優先使用
