# 07 サブエージェント

Claude Codeのサブエージェント（Taskツール）機能を体験する。
複雑なタスクを専門エージェントに委譲できる。

## お題

以下の調査・分析タスクをClaude Codeに依頼しよう。
Claude Codeが自動でサブエージェントを起動して並列処理する。

1. 「このリポジトリの全ファイルを調査して、各ディレクトリの目的をまとめて」
2. 「Pythonのfeedparserライブラリのドキュメントを調べて使い方をまとめて」
3. 「このプロジェクトのコードを全体的に分析して、改善提案をリストアップして」

## サブエージェントの種類

| エージェント | 用途 |
|---|---|
| `Bash` | コマンド実行専門 |
| `Explore` | コードベース探索 |
| `Plan` | 実装計画立案 |
| `general-purpose` | 汎用調査・複数ステップタスク |

## 特徴

- 独立したタスクは並列実行で高速化
- メインのコンテキストウィンドウを節約できる
