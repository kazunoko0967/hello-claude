# Hello Claude

Claude Code の主要機能を体験するための Hello World 演習集です。

## 概要

各ディレクトリに独立した演習が用意されています。
Claude Code に話しかけながら、機能を一つずつ試していきましょう。

## 演習一覧

| ディレクトリ | テーマ | 内容 |
|---|---|---|
| `01_file_operations` | ファイル操作 | ファイルの読み取り・バグ修正・関数追加・新規作成 |
| `02_git_operations` | Git操作 | status確認・diff・コミット・ログ・PR作成 |
| `03_web_search` | Web検索・取得 | WebSearch / WebFetch で情報収集してコードに反映 |
| `04_bash_commands` | Bashコマンド実行 | テスト実行・スクリプト実行・プロセス確認 |
| `05_code_review` | コードレビュー | 問題点指摘・修正・セキュリティチェック |
| `06_plan_mode` | Plan Mode | 実装方針を確認・承認してから実装 |
| `07_subagents` | サブエージェント | 複雑なタスクを専門エージェントに委譲・並列実行 |
| `08_hooks` | Hooks | ツール実行前後に自動でシェルコマンドを実行 |
| `09_skills` | Skills（スラッシュコマンド） | カスタムコマンドを作成して再利用 |
| `10_mcp` | MCP Servers | 外部ツール・サービスをClaude Codeに接続 |

## 使い方

1. 各ディレクトリの `README.md` を開いてお題を確認する
2. Claude Code を起動する
3. お題の指示文をそのまま Claude Code に投げてみる
