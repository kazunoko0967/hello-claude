# 06 Plan Mode

Claude Codeのプランニング機能を体験する。
実装前に設計を確認・承認できる。

## お題

以下の機能追加をClaude Codeに依頼してみよう。
Claude Codeが自動でPlan modeに入り、実装方針を提示してくれる。

1. 「`todo_app.py` にタスクの優先度機能を追加して」
2. 「TodoAppにデータのCSVエクスポート機能を追加して」
3. 「TodoAppのデータをJSONファイルに永続化する機能を追加して」

## Plan Modeの流れ

1. Claude Codeがコードを調査
2. 実装方針をリストアップ
3. **ユーザーが承認** → 実装開始
4. ユーザーが修正を指示 → 方針を変更してから実装

## 特徴

複数ファイルに影響する変更や、複数の実装方針が考えられる場合に自動でPlan modeに入る。
