# 09 Skills（スラッシュコマンド）

Claude Codeのカスタムスラッシュコマンド機能を体験する。
よく使う操作をコマンド化して再利用できる。

## 設定場所

`.claude/commands/` ディレクトリ内に Markdown ファイルを置く。

## 組み込みスキル（このリポジトリで使えるもの）

| コマンド | 用途 |
|---|---|
| `/commit` | git commitメッセージを自動生成してコミット |
| `/code-review` | PRをコードレビュー |
| `/init` | CLAUDE.mdを生成 |

## お題

1. **組み込みコマンド**: `/commit` を実行してみる
2. **カスタムコマンド作成**: `hello_skill.md` を参考にカスタムスキルを作る
3. **スキル実行**: 作成したスキルを `/hello` で呼び出す

## カスタムスキル作成手順

1. `.claude/commands/hello.md` を作成
2. ファイルにプロンプトを記述
3. Claude Codeで `/hello` と入力して実行
