# 03 Web検索・取得

Claude CodeのWebSearch・WebFetch機能を体験する。

## お題

1. **Web検索**: 「Anthropic Claude APIの最新モデル一覧を調べて」
2. **ページ取得**: 「https://docs.anthropic.com から主要な機能を要約して」
3. **情報収集+コード**: 「PythonのAnthropicライブラリの最新バージョンを調べて、requirements.txtに追記して」

## 特徴

- WebSearchは米国からのみ利用可能
- WebFetchは認証が必要なページ（Google Docs等）は取得不可
